## Personal website

This is my [personal website](https://MacTeefey.github.io), built with Jekyll and incorporating design elements from the [HTML5 UP Phantom template](https://html5up.net/phantom).

### About

This website showcases my research, projects, and educational background.

### Template Credits

This site incorporates design elements and styling from:

**Phantom** by [HTML5 UP](https://html5up.net) | [@ajlkn](https://twitter.com/ajlkn)

Phantom is a free, fully responsive HTML5 + CSS3 site template designed by [@ajlkn](https://twitter.com/ajlkn) for [HTML5 UP](https://html5up.net) and released for free under the [Creative Commons license](https://html5up.net/license).

### Modifications

This site has been adapted from the original Phantom template to work with Jekyll:

- Converted static HTML to Jekyll layouts and includes
- Integrated with Jekyll's data files for menu configuration
- Added support for submenu navigation in the sidebar
- Customized card layouts for research document display
- Integrated with the Lagrange theme's base styling
